api externa para consulta de frete

1.  Correios (API SOAP/REST via WS)

    Descrição: Usado para cálculos de frete no Brasil. A API oficial é SOAP, mas existem wrappers REST.
    Exemplo de wrapper REST: API Correios ou serviços de terceiros.
    Bibliotecas úteis:
    correios-brasil — facilita o uso da API dos Correios.
    Recurso: Consulta de CEP, cálculo de frete, prazo de entrega.

2.  Melhor Envio (API REST)

    Descrição: Integra diversas transportadoras (Correios, Jadlog, Azul Cargo, etc.) em uma API unificada.
    Documentação: Melhor Envio API
    Recurso: Simulação de frete, cálculo de prazos, geração de etiquetas de envio.

3.  Frenet (API REST)

    Descrição: Calcula frete com múltiplas transportadoras, incluindo Correios.
    Documentação: Frenet API
    Recurso: Cálculo de frete, simulação de prazos, rastreamento.

4.  ViaCEP (API REST)

        Descrição: Consulta gratuita de endereços a partir do CEP.
        Documentação: ViaCEP
        Exemplo de uso com fetch:

        '''
        async function consultaCEP(cep: string) {

        const response = await fetch(`https://viacep.com.br/ws/${cep}/json/`);
        const data = await response.json();
            console.log(data);
        }

        consultaCEP('01001000');
